Lets first see the overview of the following -



1. Webservers

A web server stores and provides data for a web application or a website to the client which has requested for the data A web server communicates with the browser using a protocol named HTTP also known as Hypertext Transfer protocol . In today's world most of the website uses this protocol or on HTTPS where S stands for secure.

  1. HTTP sends data over Port 80.
  2. HTTPS sends data over Port 443.

2. Web Service

The term Web service(WS) is either:

1.A service offered by an electronic device to another electronic device, communicating with each other via the World Wide Web.
2.A server running on a computer device, listening for requests at a particular port over a network, serving web documents and creating web applications services, which serve in solving specific domain problems over the Web (WWW, Internet, HTTP).

3. Common Object Request Broker Architecture (CORBA)

The Common Object Request Broker Architecture (CORBA) is a specification developed by the Object Management Group (OMG). CORBA describes a messaging mechanism by which objects distributed over a network can communicate with each other irrespective of the platform and language used to develop those objects.
